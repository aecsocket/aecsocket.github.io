ArenaKey

@JvmInline
value class ArenaKey(val self: Long)

A key used in a GenArena, which is a wrapper around a Long. The upper 32 bits represent the index at which this key points to (index), and the lower 32 bits represent the generation of the item this key was made for (gen).
