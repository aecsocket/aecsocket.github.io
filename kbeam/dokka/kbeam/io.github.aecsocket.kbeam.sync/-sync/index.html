Sync

interface Sync<T : Any>

A locked resource on which reads and writes must be done only under the protection of a lock.

To access the underlying T, using withLock is preferred; if that is not possible, use lock or tryLock, then manually unlock afterward. A typical usage pattern is:

fun addOne(res: Sync<Int>): Int {
  return res.withLock { value ->
    value + 1
  }
}

fun addOne(res: Sync<Int>): Int {
  val value: Int = res.lock()

    return value + 1
  } finally {

  }
